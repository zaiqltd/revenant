About

Every chip written from scratch in Rust → WebAssembly: SM83 CPU stepped on the 4.194304 MHz clock, a true pixel-FIFO PPU (acid2 pixel-perfect), 4-channel APU, MBC1/2/3/5 + RTC, CGB double-speed & HDMA. Identical input → byte-identical frames.
Run locally: cd web && python -m http.server 8080 → open http://localhost:8080. Battery saves persist in your browser per-cartridge. The accuracy scoreboard lives in SCORECARD.md.
